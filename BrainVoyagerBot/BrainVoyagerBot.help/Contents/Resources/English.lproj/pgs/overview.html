Introduction

What Does BrainVoyagerBot Do?

BrainVoyagerBot is an application that includes four utilities for automating (or helping to automate) file creation and file processing in BrainVoyager QX:

  • List Generator: generates a text file that lists paths to FMR files and corresponding target FMR files in the selected folders. The text file can be loaded in BrainVoyager's Batch Processing Wizard instead of selecting the files one by one in the Wizard (as would otherwise be necessary).
  • FMR Maker: automatically creates FMR files for up to 100 subjects at once. Requires EPI/dicom image format.
  • FMR Helper: alternative to FMR Maker for data sets with 3mm-inplane resolution. Generates a text file with paths to all files necessary for FMR creation. The text file can then be used with "FMR Maker.js" (included in BrainVoyagerBot), which can be run from within BrainVoyager to read the text file and automatically create FMR files.
  • VTC Helper: similarly to FMR Helper, VTC Helper generates a text file that can be used with the included "VTC Maker.js" file to automatically create VTC's.

Running BrainVoyagerBot's JavaScript Files from BrainVoyager QX

Once BrainVoyagerBot is installed (see Installing BrainVoyagerBot for details), the two JavaScript files, "FMR Maker.js" and "VTC Maker.js," can be run from BrainVoyager by opening the Scripts > Edit and Run Scripts... menu (NOTE: if BrainVoyager is already running when you start FMR Helper or VTC Helper, the Edit and Run Scripts pane will open automatically). Double-click the name of the script in the left-hand side bar, then paste the path as directed in the instrucitons for FMR Helper and VTC Helper.
